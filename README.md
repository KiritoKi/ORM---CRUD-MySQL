## How to Install

### Backend (API)

* To download the project follow the instructions bellow:

```
1. git clone https://github.com/KiritoKi/ORM---CRUD-MySQL.git
2. cd ORM---CRUD-MySQL
```

* Install the dependencies and start the server:

```
3. npm install mysql
4. npm install express
5. npm install ejs
6. npm run start
```
